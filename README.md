# Cycle.js Demo
Experiments with using [Cycle.js](http://cycle.js.org/) and [Webpack](http://webpack.github.io/).

## Installation

```
npm install
```

## Development mode

```
npm start
```

## Release for production

```
npm run release
```

Outputs files to `./build`.
